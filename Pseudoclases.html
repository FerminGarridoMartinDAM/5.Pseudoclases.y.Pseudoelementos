5. Pseudoclases y Pseudoelementos

    Pseudoclases y Pseudoelementos

    Los pseudoelementos y pseudoclases en CSS son herramientas que nos permiten aplicar estilos a partes específicas de un elemento HTML, ya sea en respuesta a interacciones del usuario o a la estructura del documento.

  • Pseudoclases

    Las pseudoclases son un tipo de selector en CSS que permite aplicar estilos a un elemento en función de su estado o posición en el documento. Se definen utilizando dos puntos (:) antes del nombre de la pseudoclase.

    • Pseudoclases:active

      La pseudoclase :active nos permite recuperar un elemento que está siendo activado por el usuario (normalmente, durante el click de un ratón hasta soltarlo). Esta pseudoclase se utiliza mucho en los elementos anchor y button.

      Ejemplo Pseudoclases:active

      De esta manera el botón al estar pulsado convertirá las letras (del botón) en naranja. button:active { color: orange; font-weight: bold; }

    • Pseudoclases:any-link

      La pseudoclase :any-link nos permite recuperar un elemento link que haya sido visitado o no, siempre y cuando tenga un valor en su href.

      Pseudoclases:any-link ejemplo:

      a:any-link { color: purple; font-weight: bold; } lo modificamos por a[class="any"]:any-link { color: purple; font-weight: bold; } para que solo actue en este apartado y no en todo el documento.º de esta manera los enlaces que tengan un valor en su atributo href se mostrarán en color púrpura y en negrita.

    • Pseudoclases:checked

      La pseudoclase :checked nos permite recuperar cualquier elemento checkbox, radio u option que esté marcado o con el atributo checked activo.

      Ejemplo Pseudoclases:checked

      input:checked + label { color: crimson;} Con este codigo cambiamos el color o formato al raido que este marcado.

    • Pseudoclases:default

      La pseudoclase :default nos permite recuperar cualquier elemento que esté en su estado predeterminado dentro de un formulario. Esta pseudoclase se utiliza mucho en elementos button, checkbox, radio y option que tengan una opción marcada desde el principio.

      Ejemplo Pseudoclases:default

      el CSS: input:default + label { color: green; } con este codigo cambiamos el color o formato al elemento que este en estado predeterminado en un formulario.

    • Pseudoclases:disabledtitulo

      La pseudoclase :disabled nos permite recuperar un elemento deshabilitado o con el atributo disabled activo.

      Ejemplo Pseudoclases:disabled

      en este ejemplo vamos a marcar el boton que esta Desactivado en color rojo. button:disabled { color: red; }

    • Pseudoclases:enabled

      La pseudoclase :enabled nos permite recuperar cualquier elemento habilitado en el documento, diferenciando asi a los deshabilitados. Enabled en CSS se utiliza para seleccionar elementos de formulario que están habilitados, es decir, aquellos con los que el usuario puede interactuar. Es especialmente útil cuando se quiere aplicar estilos visuales distintos a los elementos activos frente a los deshabilitados (:disabled).

      Ejemplo Pseudoclases:enabled

      En este ejemplo vamos a recupera "cambiar" el elemento que esta habilitado o con el atributo enabled activo y lo vamos a poner de color verde y el disabled en rojo. He cambiado un poco el codigo añadiendole a las etiquetas button la clase ejemploenabled para que solo cambie los botones de este ejemplo y no la de las anteriores , por tanto tambien he añadido al codigo un selector de atributo class="ejemploenabled" para que no afecte a los botones de arriba. que se añade con < class="ejemploenabled">

      button[class="ejemploenabled"]:disabled { color: red; }

      button[class="ejemploenabled"]:enabled { color: green; }

    • Pseudoclases:first-child

      La pseudoclase :first-child nos devuelve el primer elemento de un grupo de elementos hermanos con el mismo padre.

      Ejemplo Pseudoclases:first-child

      En este ejemplo vamos a recuperar el primer hijo de la lista y vamos a cambiarle el color

      li:first-child { color: crimson; } !Y yo le he añadido[class="primer-hijo"] que es una clase que he creado para estos "li" que me coja el primer hijo de esa clase y no de todo el documento, por tanto: li[class="primer-hijo"]:first-child { color: crimson; }

      Pseudoclases:focus

      La pseudoclase :focus representa los elementos que han activado su foco al usuario interactuar con ellos o al haber entrado en uno de los elementos.

      Ejemplo Focus

      En este caso vamos a hacer que cuando interactuamos con un elemento se cambie el color del background para que se resalte.

      input:focus { background-color: burlywood; }

    • Pseudoclases:hover

      La pseudoclase :hover nos permite recuperar un elemento que ha sido activado por el usuario durante el desplazamiento del cursor sobre él. No es necesario que el usuario active el elemento para que esta pseudoclase sea activada.

      Ejemplo Pseudoclases:hover

      En este caso cuando pasemos el cursor sobre la zona del div class="hover" se cambiara el color de fondo de violeta que lo hemos puesto por defecto a rojo

      div[class="hover"] { width: 100px; height: 100px; background-color:blueviolet; } div[class="hover"]:hover { background-color: crimson; }

    • Pseudoclases:invalid

      La pseudoclase :invalid nos permitirá recuperar un elemento que haya tenido un error de campo a la hora de validarlo dentro de un formulario.

      Ejemplo Pseudoclases:invalid

      Aqui vamos a hacer que cuando el elemento tenga un error se nos marque el fondo en rojo.(En este caso al ser un input required y todavia no estar completado, se nos marca con invalido con el fondo rojo) le añadimos clase para que no nos modifique toda la pagina. class="emailinvalid"

      input[class="emailinvalid"]:invalid { background-color: crimson; }

    • Pseudoclases:last-child

      La pseudoclase :last-child nos devuelve el último elemento de un grupo de elementos hermanos con el mismo padre.

      Ejemplo Pseudoclases:last-child

      Aqui vamos a recuperar el ultimo hermano de la lista y cambiarle el color a crimsom. Le hemos añadido la class="ultimo-hijo" para que no afecte a todos los li del documento y solo a los de este ejemplo.

      li[class="ultimo-hijo"]:last-child { color: crimson; }

    • Pseudoclases:not

      La pseudoclase :not nos devuelve los elementos que no coincidan con el selector indicado, es decir, negamos los elementos. Se le conoce también como la pseudoclase de negación.

      Ejemplo Pseudoclases:not

      En este ejemplo nos marcaria todo lo que no fuera h1, pero hemos puesto el codigo para que solo ataque a la class="not"

      main :not(h1) { color: orange; } .not:not(h1) {color: orange; }

    • Pseudoclases:nth-child

      La pseudoclase :nth-child nos devuelve los elementos que coincidan con la posición indicada entre una lista de elementos hermanos. Esta posición se le indicará mediante el número, odd (impar) o even (par).

      Ejemplo nth-child

      Aqui vamos a cambiar el 3er elemento de la lista a color verde para que sea solo el 3er elemento de esta lista le añadimos la clase li class="nth" li[class="nth"]:nth-child(3) { color: orange; }

      Ejemplo Pseudoclases:nth-last-child

    • Pseudoclases:read-only

      Ejemplo Pseudoclases:read-only

      La pseudo-clase :read-only nos devuelve los elementos que no son editables por el usuario, normalmente son aquellos inputs que incluyen el atributo “read-only” o etiquetas normales no editables.

      aqui vamos a cambiarle cambiar el color a rojo de un campo de solo lectura input:read-only color: crimson; input { width: 100%; }

    • La pseudo-clase :read-write nos devuelve los elementos editables por el usuario, normalmente suele aplicarse a inputs o etiquetas que incluyan el atributo “contenteditable”.

      Ejemplo Pseudoclases:read-write

      Aqui atacaremos la parte editable y la pondremos de color verde input:read-write color: green; input { width: 100%; }

    • La pseudo-clase :required nos devuelve los elementos que contengan el atributo “required”, normalmente se aplica a las etiquetas “input”.

      Ejemplo Pseudoclases:required

      Aqui vamos a atacar los campos requeridos cambiandoles el color y el background input[obligatorio]:required::placeholder { color: crimsom; } input[obligatorio]:required::placeholder {background-color: yellow} input { width: 100%; }

    • La pseudo-clase :root nos devuelve el elemento raíz del documento para tener una mayor especificidad incluso que la etiqueta html. Se utiliza mucho para declarar variables en nuestra hoja de estilos.

      Ejemplo Pseudoclases:root

      Aqui le damos estilo a toda la pagina con esas reglas y parametros. :root{ font-size: 16px; font-family: "Roboto",sans-serif; --primary: crimson; --secondary: red; --background: grey; --border-radius: 1.2rem; }

    • ::after crea un pseudoelemento, por defecto in-line, después del elemento a través de la propiedad “content”. Estos pseudoelementos serán contenidos por la caja del elemento.

      Ejemplo Pseudoelementos::after

      Vamos a crear un elemento despes del elemento que contenga li en este caso un check, con un padding a la izquerda de 3px li::after { content: "✅"; padding-left: 3px; }
